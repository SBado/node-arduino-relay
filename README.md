# node-arduino-rele

This is the source code for a simple example using Nodejs and Socket.io 
for activate relé from a Arduino microcontroller.


for install Nodejs
 - http://nodejs.org


Arduino modules used
 - Arduino UNO R3
 - Ethernet R3 Arduino Shield
 - rele module (https://www.robocore.net/modules.php?name=GR_LojaVirtual&prod=258)
